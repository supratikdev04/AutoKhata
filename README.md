AutoKhata – Expense Tracker Web Application :-
==============================================
AutoKhata is a full-stack expense tracking web application built to simplify and automate personal expense management. The project focuses on reducing manual data entry by allowing users to upload scanned bills and receipts, automatically extracting total amounts using OCR, and organizing expenses into meaningful categories.
The application is designed to handle real-world financial data securely while providing clear insights through filtering, summaries, and downloadable reports.

✨ Key Highlights :-
=====================
>> Automates expense entry using bill scanning and OCR.
>> Ensures accurate and structured financial record-keeping.
>> Provides downloadable reports for easy review and sharing.
>> Built with a scalable backend and a responsive frontend.

🚀 Features :-
===============
🔹 Expense Management:-
------------------------
> Add expenses manually or via scanned bills.
> Automatically extract total amounts from bills using OCR.
> Categorize expenses such as Food, Electricity, Internet, Travel, etc.
> Edit or delete existing expense records.

🔹 Filtering & Analysis:-
--------------------------
> Filter expenses by date range and category.
> View organized expense lists for better tracking.
> Generate structured reports for selected time periods.

🔹 Reports & Export:-
-----------------------
> Download expense reports in PDF format.
> Well-formatted reports suitable for personal or professional use.

🔹 Authentication & Security :-
--------------------------------
> Secure user authentication.
> Each user can access only their own expense data.
> Backend validation to maintain data accuracy.

🔹 User Experience :-
----------------------
> Clean and responsive UI.
> Optimized forms and layouts for desktop usage.
> Simple workflow focused on ease of use.

🛠 Tech Stack :-
================
>> Frontend [ HTML,CSS,JavaScript ].
>> Backend [ Flask (Python) , RESTful APIs ]. 
>> Database [PostgreSQL / Supabase].
>> Additional Tools :-
    > OCR for bill text extraction.
    > PDF generation for downloadable reports.

🧠 Why This Project?
=====================
Managing expenses manually is time-consuming and error-prone, especially when dealing with multiple bills and receipts. AutoKhata aims to solve this problem by combining automation (OCR), structured data storage, and reporting features into a single web application.

This project demonstrates real-world backend development, database design, and integration of third-party tools to solve a practical problem.
